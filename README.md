# Pendu - Games


## Project Setup

first clone this repo

```sh
git clone https://github.com/Nasan01/pendu-game.git
```

then

```sh
cd pendu-game
```
and Install it's dependencies

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
